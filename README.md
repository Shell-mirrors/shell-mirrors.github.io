# Shell Mirrors

This is a shell script repository.

## Repository Content:

1. Shadowsocks for python version install Tools:

``` bash
$ wget script.xmxin.top/Shadowsocks_Install_Tools.sh -O SSTools.sh && bash SSTools.sh
```

2. BBR Tools:

``` bash
$ wget script.xmxin.top/bin/tcp_bbr -O /bin/tcp_bbr && chmod +x /bin/tcp_bbr
```

3. Git Tools:

``` bash
$ wget script.xmxin.top/bin/gtools -O /bin/gtools && chmod +x /bin/gtools
```

4. SSserver Tools:

``` bash
$ wget script.xmxin.top/bin/sss -O /bin/sss && chmod +x /bin/sss
```

5. SHC and GZEXE Tools:

``` bash
$ wget script.xmxin.top/Shc_Gzexe_Tools.sh && bash Shc_Gzexe_Tools.sh
```

6. SeVPN Tools:

``` bash
$ wget script.xmxin.top/SeVPN.sh -O sevpn && bash sevpn
```

7. SSR Install:

``` bash
$ wget script.xmxin.top/ssr.sh && bash ssr.sh
```
8. VNC Install:

``` bash
$ wget script.xmxin.top/LXDE_VNC_Install.sh -O Install.sh && bash Install.sh
```
